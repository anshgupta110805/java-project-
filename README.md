# 🏦 Mini Bank System v1.0

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Status](https://img.shields.io/badge/Status-Complete-green?style=for-the-badge)

A robust and secure console-based Mini Bank System built using Java. This project demonstrates core Object-Oriented Programming (OOP) concepts like Encapsulation, Constructor overloading, and dynamic data handling with ArrayLists.

---

## 🚀 Features

- **🔐 Secure Login**: Access accounts using unique Account Numbers and PINs.
- **💰 Balance Management**: Real-time balance updates for deposits and withdrawals.
- **📜 Transaction History**: Track every action (deposits, withdrawals, and account opening).
- **🔑 Security**: Change your transaction PIN anytime.
- **⚡ User-Friendly Menu**: Easy-to-navigate console interface.

---

## 🛠️ Tech Stack

- **Language**: Java
- **Concepts Used**:
  - Class & Objects
  - Encapsulation (Private fields with getter/setter logic)
  - Collections (ArrayList for transaction logs)
  - Control Flow (Switch-case, Do-While loops)

## 🚦 How to Run

1. **Prerequisites**: Make sure you have [JDK](https://www.oracle.com/java/technologies/downloads/) installed.
2. **Clone the Repo**:
   ```bash
   git clone https://github.com/anshgupta110805/mini-bank-system-.git
   cd mini-bank-system-
   ```
3. **Compile**:
   ```bash
   javac MiniBankSystem.java
   ```
4. **Run**:
   ```bash
   java MiniBankSystem
   ```
